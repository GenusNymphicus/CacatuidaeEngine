Detailed Description

Interface for the renderer-implementations Provides an interface for the various renderer-implementations.

A Renderer is the interface between the GraphicsLibraries(OpenGL, DirectX, ..) and the outside. Ideally, no other file will contains the graphicslibrary-specific code, except for the renderer-implementations.
